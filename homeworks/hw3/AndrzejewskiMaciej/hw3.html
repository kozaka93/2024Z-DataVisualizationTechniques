Wizualizacja rdowa

rdo: Post Tomasza Sakiewicza na platformie X
Data publikacji: 10:00 AM 6 lis 2024

Celem wizualizacji byy pokazanie odniesionego przez TV Republik sukcesu ogldalnociowego w czasie wieczoru wyborczego 05.11.2024 (wybory prezydenckie w USA).

Elementy do poprawy

Gwnym problemem tego wykresu jest maa czytelno.

Co naleaoby poprawi:

  • oznaczenia na osi OY - s to liczby zapisane przy pomocy nieoddzielonej w aden sposb duej liczby zer, co utrudnia dokadne odczytanie rzdu wielkoci tych liczb (zwaszcza podczas wywietlania tego wykresu w telewizji, gdzie nie da si przybliy obrazu);

  • oznaczenia na osi OX - reprezentuj mao czytelne przedziay godzinowe, a dodatkowo zawieraj zbdne (zmniejszajce czytelno) tytuy programw, ktre dotycz wycznie jednej stacji; pojawia si take enigmatyczna godzina 25:00:00;

  • legenda - zostaa stworzona zbyt maa, dodatkowo wystpuj w niej zbdne informacje (zamiast samych nazw stacji), a take nachodz na ni znaczniki z osi OX;

  • tytu - osadzony poza kontekstem przecitnemu odbiorcy nie mwi nic o treci wizualizacji;

  • wyrnienie fragmentu, na ktrym nam zaley - tego po prostu brakuje.

Nowa wizualizacja

Poniej znajduje si nowa wersja wykresu (oraz odpowiedni kod), stworzona za pomoc ggplot2 oraz danych podobnych do pierwotnych (w kluczowych fragmentach odczytanych z wykresu, wpp. wygenerowanych pseudolosowo).

df_long <- df %>% mutate(X = factor(X, levels = X)) %>% 
  tidyr::pivot_longer(cols = 2:7) %>% 
  rename(time = X, station = name, count = value)

ggplot(df_long,
       aes(time, count, group = station, color = station)) +
  geom_line(linewidth = 1.5, alpha = 0.7) +
  labs(title = "Wykres ogldalnoci poszczeglnych stacji dnia 05.11.2024",
       y = "Liczba osb ogldajcych dany kana",
       x = "Godzina",
       color = "Stacja") +
  theme_minimal() +
  theme(axis.text.x = element_text(size = 15, angle = 45, hjust = 1, face = "bold"),
        axis.text.y = element_text(size = 15, face = "bold"),
        plot.title = element_text(size = 15),
        legend.text = element_text(size = 15),
        legend.title = element_text(size = 15),
        axis.title = element_text(size = 15)
        ) +
  scale_x_discrete(breaks = unique(df_long$time)[
    seq(1, length(df_long$time), by = 3)]) +
  scale_color_manual(values = brewer.pal(6,"Set1")[c(2,1,3:6)],
                     labels = c("Polsat News", expression(underline("TV Republika")), "TVN 24", "TVP Info", "Wpolsce24", "Wydarzenia 24")) +
  scale_y_continuous(labels = scales::label_number(big.mark = " ")) +
  geom_vline(xintercept = 52, color = "red", linetype = "dashed", linewidth = 1) +
  geom_vline(xintercept = 61, color = "red", linetype = "dashed", linewidth = 1) +
  annotate("text", x = 58, y = 900000, 
           label = "Wieczr\nwyborczy",
           color = "red",   
           size = 5,
           fontface = "bold", 
           family = "serif", 
           hjust = 0.5, vjust = 0.5)

Uzasadnienie poprawy

Nowa wersja wykresu jest lepsza od oryginau, poniewa:

  • liczby na osi OY s bardziej czytelne;

  • liczba znacznikw na osi OX jest mniejsza i zawiera tylko godzin (kluczow informacj);

  • tytu oraz legenda s przystpniejsze w odbiorze;

  • fragment wykresu, ktry nas interesuje, zosta wyranie zaznaczony.
