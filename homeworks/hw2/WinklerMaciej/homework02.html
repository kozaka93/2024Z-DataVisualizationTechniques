Zadanie 1

Jak wyglda rozkad ceny nieruchomoci w poszczeglnych kodach pocztowych na przestrzeni lat? Rozwaamy kody pocztowe, ktre maj wicej ni 280 posiadoci.

# Tu umie kod do wygenerowania odpowiedzi do zadania.

library(scales) #wczytujemy bibliotek scales dla wyraniejszego odczytywania wykresu
## Warning: pakiet 'scales' zosta zbudowany w wersji R 4.3.3
result01 <- df %>% 
  mutate(date = as.Date(date, format = "%Y%m%dT%H%M%S"),
         year = format(date, "%Y")) %>%
  group_by(zipcode) %>% 
  filter(n() > 280) %>% 
  ggplot(aes(x = factor(zipcode), y = price, fill = factor(year))) +
  geom_boxplot() +
  labs(title = "Rozkad ceny nieruchomoci na przestrzeni lat",
       x = "kod pocztowy", y = "cena",
       fill = "rok") +
  scale_y_continuous(labels = label_number(scale = 1e-3, suffix = "K")) + #dla janiejszego przedstawienia cen K = 1000
  theme_minimal() +
  theme(axis.text.x = element_text(angle = 45, hjust = 1))
result01

Komentarz: #Rozkad cen zosta ukazany na wykresie pudekowym stworzonym powyej. Mona zauway, #e ceny byy nieco wysze w roku 2015 dla wikszoci mieszka o konkretnych kodach pocztowych.

## Zadanie 2

Czy jest rnica w rozkadzie ceny nierchomoci od powierzchni mieszkalnej uwzgldniajc podzia na nieruchomoci z piwnic i bez?

## Zadanie 5 Rozwamy tylko nieruchomoci, ktre miay remont. W ktrej dekadzie byo najwicej przeprowadzonych remontw? Czy w poszczeglnych dekadach jest porwnywalny procent nieruchmoci dla kadej wartoci miary jakoci i wykoczenia nieruchomoci?

# Tu umie kod do wygenerowania odpowiedzi do zadania.

dfx <- df %>% #Tworzymy ramk pomocniczm ktrej uyjemy do znalezienia odpowiedzi na pytanie oraz stworzenia wykresu.
  select(id, yr_renovated, grade) %>% 
  filter(yr_renovated != 0) %>% 
  mutate(decade = floor(yr_renovated / 10) * 10)

dfy <- dfx %>% #Odpowied do pytania "W ktrej dekadzie byo najwicej przeprowadzonych remontw?"
  group_by(decade) %>% 
  summarise(dec_renov = n()) %>% 
  filter(dec_renov == max(dec_renov))
dfy #Wywoujc ramk danych dfy moemy si przekona e to w dekadzie 2000 - 2010 byo najwicej przeprowadzonych remontw - 272.
## # A tibble: 1  2
##   decade dec_renov
##    <dbl>     <int>
## 1   2000       272
result05 <- dfx %>%
  group_by(decade, grade) %>%
  summarise(count = n()) %>%
  ungroup() %>%
  group_by(decade) %>%
  mutate(percentage = count / sum(count)) %>% 
  ggplot(aes(x = factor(decade), y = percentage, fill = factor(grade))) +
  geom_bar(stat = "identity", position = "fill") +
  labs(
    title = "Procentowy udzia nieruchomoci wedug jakoci wykoczenia w poszczeglnych dekadach",
    x = "Dekada",
    y = "Procentowy udzia",
    fill = "Jako"
  ) +
  scale_y_continuous(labels = scales::percent) +
  theme_minimal()
result05

Komentarz: #Najwicej przeprowadzonych remontw odbyo si w dekadzie 2000-2010, byo ich 272. Z kolei wykres #stworzony powyej pokazuje nam, e procenty nieruchomoci ze wzgldu na miar jakoci rni si w poszczeglnych dekadach. #Najbardziej podobne wartoci moemy dostrzec w jakoci 7 dla wszystkich dekad poza 1930 - 1940.

## Zadanie 6

Jak prezentuje si warto minimalna, rednia i maksymalna ceny nieruchmoci w podziele na rok budowy nieruchomoci.

# Tu umie kod do wygenerowania odpowiedzi do zadania.

result06 <- df %>% 
  select(id, price, yr_built) %>% 
  unique() %>% 
  group_by(yr_built) %>% 
  summarise(min = min(price), max = max(price), mean = mean(price)) %>% 
  ggplot(aes(x = yr_built, y = price)) +
  labs(title = 'Wykres wartoci minimalnych, maksymalnych i rednia cen nieruchomoci w podziele na rok budowy',
       x = 'rok budowy',
       y = 'cena',
       color = 'warto') +
  geom_line(aes(y = min, color = "Min")) +
  geom_line(aes(y = max, color = "Max")) +
  geom_line(aes(y = mean, color = "rednia")) +
  scale_y_continuous(labels = scales::label_number(scale = 1e-3, suffix = "K")) +
  scale_color_manual(values = c("Min" = "blue", "Max" = "red", "rednia" = "green"))
result06

Komentarz: #Wartoci minimalne, maksymalne oraz warto rednia zostay ukazane na wykresie stworzonym powyej. #Mona zauway, e wartoci naksymalne cen nieruchomoci z poszczeglnych lat budowy drastycznie si zwikszaj #lub zmniejszaj w kolejnych latach, za to wartoci minimalne oraz rednie utrzymuj podobn warto we wszystkich latach.
